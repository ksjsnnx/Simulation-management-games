# 模拟经营项目（Unity）- 脚本说明与使用指南

这是一个基于 Unity 的模拟经营练习项目，核心玩法包含：
- 购买土地、放置建筑、生产与收获
- 背包与商品流通
- 主线/支线任务推进与等级成长
- NPC 对话与任务触发
- 本地存档（新建存档 + 上次游戏）




## 项目亮点

- **模块化管理器架构**：`GameManager`、`UIManager`、`TaskManager`、`BuildController` 等职责清晰。
- **数据驱动**：通过 JSON 表初始化建筑、产物、对话、任务、商店等配置。
- **事件系统解耦**：通过 `EventCenter` 广播等级变化、金币变化、任务状态等事件。
- **完整经营循环**：土地状态流转（未购买 -> 已购买 -> 已建造）+ 建筑产出 + 任务反馈。
- **可扩展性较好**：新增建筑/任务/对话多数可通过配置数据接入。

## 目录结构（Scripts）

```text
Assets/Scripts
├─ Archive/        # 输入、存档相关
├─ Build/          # 建筑与产物逻辑（含具体建筑类型）
├─ Controller/     # 时间、建造、相机等控制器
├─ Data/           # 核心数据结构（PlayerData、TaskItemData 等）
├─ Dialogue/       # 对话项逻辑
├─ Enum/           # 枚举定义（GameEvent 等）
├─ Json/           # JSON 读写与转换（含 LitJson）
├─ Manager/        # 各类管理器（Game/UI/DB/Scene/Event 等）
├─ Task/           # 任务系统（主线/支线）
└─ UIPanel/        # 各 UI 面板脚本
```

## 核心系统说明

### 1) 游戏初始化与总控
- `GameManager` 负责启动流程、数据初始化、时间系统初始化、存档保存。
- 启动后默认打开 `StartPanel`，从新游戏或读档进入主场景。
- 初始化内容包括：
  - 建筑数据、产物数据、消耗表、商店表
  - 对话数据、任务数据、玩家数据、背包数据

### 2) UI 系统
- `UIManager` 通过泛型 `openPanel<T>() / closePanel<T>()` 管理面板生命周期。
- UI 预制体通过 `Resources/UI/UIPanel/` 路径加载。
- 常见面板：开始菜单、背包、建造详情、任务面板、对话面板、暂停面板等。

### 3) 建造与土地系统
- `BuildController` 负责地块选择、点击交互、地块状态切换、建造对象管理。
- 土地状态：
  - `0` 未购买
  - `1` 已购买（可建造）
  - `2` 已建造（可查看详情/拆除）
- 通过 `currentBuildingDict` 管理场景中的建筑实例，支持增删与事件通知。

### 4) 任务系统
- `TaskManager` 管理主线与支线任务刷新、领取、完成奖励和等级推进。
- 主线通常按等级解锁，支线可领取后触发。
- 任务完成后触发奖励逻辑，并进行等级可提升检查。

### 5) 对话系统
- `DialoguePanel` 负责对话文本播放、选项分支、下一句推进。
- 对话可绑定任务 ID，用于触发支线任务领取。

### 6) 存档系统
- `GameManager.SaveArchive()` 支持两类存档：
  - 上一次游戏：`Archive/LastGame/`
  - 新建存档：`Archive/EveryGame/<ticks>/`
- 存档内容包括：玩家、背包、时间戳、地块状态、建筑状态、任务状态等。

### 7) 登录注册（可选）
- `LoginPanel` / `RegisterPanel` + `DBManager` 提供 MySQL 登录注册流程。
- 默认连接参数位于 `DBManager`（本地 `127.0.0.1`、数据库 `mygamedb`）。
- 仅在需要账号系统演示时启用；纯单机玩法不依赖数据库。

## 运行与体验

### 开发环境建议
- Unity（建议使用项目原始版本或同代 LTS）
- .NET / C#（Unity 默认）
- 可选：MySQL（如果要测试登录注册）

### 基本流程
1. 打开项目后进入启动场景。
2. 点击“新开游戏”进入主场景（或通过“加载存档”读档）。
3. 在地块上进行购买、建造、生产、查看详情与拆除等操作。
4. 查看任务面板，完成主线/支线以提升游戏等级。

### 快捷操作（代码中可见）
- `Esc`：打开暂停面板
- 鼠标右键：关闭当前面板
- `M`：打开/关闭背包

## 关键依赖

- `DG.Tweening`（DOTween）：文本/界面动效
- `LitJson`：JSON 序列化与反序列化
- `Newtonsoft.Json`：部分数据结构转换
- `MySql.Data`：数据库连接与查询（登录注册功能）
